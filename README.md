# 拥抱AI时代：做未来教育的引路人

这是一个基于 React 和 Framer Motion 开发的现代化全屏演示文稿，旨在探讨 AI 时代下教育的本质变革、技术应用与未来能力培养。

演示地址：[ai-edu.zhaolaoban.top](https://ai-edu.zhaolaoban.top)

## ✨ 项目亮点

- **赛博科技视觉**：采用霓虹青、深邃蓝与磨砂玻璃质感设计，营造极致的未来感。
- **极致流畅动效**：基于 `Framer Motion` 实现全场丝滑的入场动画、视差滚动与交互反馈。
- **高采样率演进图**：内置高精度智力演化阶梯图，精准还原 AI 发展的历史节点。
- **智能交互体验**：侧边导航实时反馈，支持悬停预览标题；内置多个 AI 互动案例 Prompt 展示。
- **多媒体深度融合**：无缝集成了高清视频演示与架构图，增强观众的感官体验。

## 🚀 技术栈

- **框架**: [React 19](https://react.dev/)
- **动效**: [Framer Motion](https://www.framer.com/motion/)
- **构建**: [Vite](https://vitejs.dev/)
- **样式**: [TailwindCSS 4](https://tailwindcss.com/) & [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- **图表**: [Recharts](https://recharts.org/)

## 🛠️ 本地开发

1. **克隆仓库**:
   ```bash
   git clone https://github.com/zhaoxi1999/ai-edu-presentation.git
   cd ai-edu-presentation
   ```

2. **安装依赖**:
   ```bash
   npm install
   ```

3. **启动开发服务器**:
   ```bash
   npm run dev
   ```

4. **构建生产版本**:
   ```bash
   npm run build
   ```

## 🌐 部署指南

该项目可以轻松部署在 [Cloudflare Pages](https://pages.cloudflare.com/)：

1. 在 Cloudflare 控制面板点击 **"Connect to Git"**。
2. 选择此仓库并设置 **Build command**: `npm run build`。
3. 设置 **Build output directory**: `dist`。
4. 完成部署！
